Attack Flow Spaces

Attack flow spaces is a domain providing hosting for namespaces associated with Attack Flow. Currently the following namespaces are available:
  • Assets
  • VERIS (A graph recreation of the VERIS schema
  • attack-flow (A graph recreation of version 1 of the Attack Flow schema.)
If you are interested in Attack Flow and how these schemas are used, you should look into Flow Manager, the first tool available for managing Attack Flows. Flow Manager documentation can be found here
